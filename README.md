# TimedPoweroff

#### 介绍

- JavaFX写的定时关机程序，支持倒计时、计划关机、重启。
- 平台：Windows、Linux (没有MacOS，这辈子都不会买Mac的，有钱我还在这儿自己敲这破代码？？？)
- 版本：Java 11
- 缺点： **启动** 是真的慢……至少在我这3千大洋的商务本上就是这么慢。。。五六秒吧。。最快也要3秒。还有就是 **模式切换后** ， **必须手动** 刷新 **时、分、秒** 三个数据， **否则** 时间设定将出现偏差。
